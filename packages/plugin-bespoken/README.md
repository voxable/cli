## `voxable bespoken:transform PROJECT_ID`

export a Voxable project and transform it into Bespoken tests

```
USAGE
  $ voxable bespoken:transform PROJECT_ID

ARGUMENTS
  PROJECT_ID  project ID

OPTIONS
  -h, --help  show CLI help

EXAMPLE
  $ voxable bespoken:transform PROJECT_ID
```
